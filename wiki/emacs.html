1 Entering and Exiting Emacs

  • Entering: The usual way to invoke Emacs is with the shell command emacs.
  • Exiting: C-x C-c Kill Emacs

2 Basic Editing Commands

2.1 Inserting Text

To insert a non-graphic character, or a character that your keyboard does not support, first quote it by typing C-q (quoted-insert).

  • C-q followed by any non-graphic character (even C-g) inserts that character.

    e.g. C-q DEL inserts a literal 'DEL' character.

  • C-q followed by a sequence of octal digits inserts the character with the specified octal character code.

    e.g. C-q 1 0 1 B inserts ‘AB’.

  • C-x 8 RET prompts for the Unicode name or code-point of a character, using the minibuffer.

    e.g. C-x 8 RET infinity RET / C-x 8 RET 221e RET inserts the infinity sign.

2.2 Change the Location of Point

For left-to-right scripts:

C-f / RIGHTMove forward
C-b / LEFTMove backward
C-n / DOWNMove down one screen line
C-p / UPMove up one screen line
C-a / HOMEMove to the beginning of the line
C-e / ENDMove to the end of the line
M-b / C-LEFT / C-RIGHTMove backward one word
M-f / C-RIGHT / M-RIGHTMove forward one word
M-rreposition point on the left margin of the center-most text line of the window
M-<Move to the top of the buffer
M->Move to the end of the buffer
C-v PAGEDOWN NEXTScroll the display one screen forward
M-v PAGEUP PRIORScroll one screen backward
M-x goto-charRead a number n and move point to buffer position n
M-g M-g / M-gRead a number n and move point to the beginning of line number n

2.3 Erasing Text

DEL / BACKSPACEDelete the character before point
DELETEDelete the character after point
C-dDelete the character after point
C-kKill to the end of the line
M-dKill forward to the end of the next
M-DELKill back to the beginning of the previous word

2.4 Undoing Changes

C-/ / C-x u / C-_Undo

2.5 Files

C-x C-fOpen or create a file

2.6 Blank Lines

C-oInsert a blank line after the cursor
C-x C-oDelete all but one of many consecutive blank lines

3 Searching and Replacement

3.1 Basics of Incremental Search

C-sBegin incremental search
C-rBegin reverse incremental search

3.2 Regular Expression Search

C-M-sBegin incremental regexp search
C-M-rBegin reverse incremental regexp search

3.3 Replacement Commands

M-% string RET newstring RETReplace some occurrences of string with newstring
C-M-% regexp RET newstring RETReplace some matches for regexp with newstring

4 The Mark and the Region

4.1 Setting the Mark

C-SPC / C-@Set the mark at point

4.2 Commands to Mark Textual Objects

M-@Set mark after end of next word
C-M-@Set mark after end of following balanced expression
M-hMove point to the beginning of the current paragraph, and set mark at the end
C-M-hMove point to the beginning of the current defun, and set mark at the end
C-x C-pMove point to the beginning of the current page, and set mark at the end
C-x hMove point to the beginning of the buffer, and set mark at the end

4.3 Operating on the Region

C-wCut
M-wCopy
C-yPaste
M-yReplace the text just yanked with an earlier batch of killed text

5 File Handling

5.1 Visiting Files

C-x C-fVisit a file
C-x C-rVisit a file for viewing, without allowing changes to it
C-x C-vVisit a different file instead of the one visited last
C-x 4 fVisit a file, in another window
C-x 5 fVisit a file, in a new frame

5.2 Saving Files

C-x C-sSave the current buffer to its file
C-x sSave any or all buffers to their files
M-~Forget that the current buffer has been changed
C-x C-wSave the current buffer with a specified file name
M-x set-visited-file-nameChange the file name under which the current buffer will be saved

6 Using Multiple Buffers

6.1 Creating and Selecting Buffers

C-x b buffer RETSelect or create a buffer named buffer
C-x 4 b buffer RETSelect buffer in another window
C-x 5 b buffer RETSelect buffer in a separate frame
C-x LEFTSelect the previous buffer in the buffer list
C-x RIGHTSelect the next buffer in the buffer list

6.2 Listing Existing Buffers

C-x C-bList the existing buffers

6.3 Miscellaneous Buffer Operations

C-x C-qToggle read-only status of buffer
M-x rename-buffer RET name RETChange the name of the current buffer
M-x rename-uniquelyRename the current buffer by adding ‘<number >’ to the end
M-x view-buffer RET buffer RETScroll through buffer buffer

6.4 Killing Buffers

C-x k bufname RETKill buffer bufname
M-x kill-some-buffersOffer to kill each buffer, one by one
M-x kill-matching-buffersOffer to kill all buffers matching a regular expression

7 Multiple Windows

7.1 Splitting Windows

C-x 2Split the selected window into two windows, one above the other
C-x 3Split the selected window into two windows, positioned side by side
C-Mouse-2In the mode line of a window, split that window

7.2 Using Other Windows

C-x oSelect another window
C-M-vScroll the next window
Mouse-1Select the window

7.3 Displaying in Another Window

C-x 4 b bufname RETSelect buffer bufname in another window
C-x 4 C-o bufname RETDisplay buffer bufname in some window, without trying to select it
C-x 4 f filename RETVisit file filename and select its buffer in another window
C-x 4 d directory RETSelect a Dired buffer for directory directory in another window
C-x 4 mStart composing a mail message
C-x 4 .Find a tag in the current tags table, similar to M-.

7.4 Deleting and Rearranging Windows

C-x 0Delete the selected window
C-x 1Delete all windows in the selected frame except the selected window
C-x 4 0Delete the selected window and kill the buffer that was showing in it
C-x ^Make selected window taller
C-x }Make selected window wider
C-x {}Make selected window narrower
C-x -Shrink this window if its buffer doesn’t need so many lines
C-x +Make all windows the same height
